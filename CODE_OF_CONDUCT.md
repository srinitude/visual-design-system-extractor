# Code of Conduct

This project is intended to be a practical, respectful space for improving an open-source Agent Skill.

## Expected Behavior

- Be respectful and constructive.
- Keep feedback specific and actionable.
- Assume good intent, but respond to impact when something causes harm.
- Keep discussions focused on the project and its users.

## Unacceptable Behavior

- Harassment, threats, or discriminatory language.
- Publishing private information without permission.
- Repeated off-topic disruption.
- Deliberate misuse of the project to expose private screenshots, credentials, or proprietary material.

## Enforcement

Maintainers may remove comments, close issues, block participants, or reject contributions that violate this code of conduct.

Report concerns through the repository maintainer's GitHub profile or, when sensitive, through the security reporting guidance in [SECURITY.md](SECURITY.md).
